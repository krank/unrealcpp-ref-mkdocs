# USphereComponent

En [UPrimitiveComponent](./README.md) som beskriver en sfär.

## SetSphereRadius()

Ändrar sfärens radie.

```cpp
SphereComponent->SetSphereRadius(SphereSize);
```

## InitSphereRadius()

Ändra sfärens radie, utan att trigga kollisioner etc.

```cpp
SphereComponent->InitSphereRadius(SphereSize);
```
