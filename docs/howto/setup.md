# Setup

* Installera **Unreal Engine 5.x** (c:a 30gb)
* Installera **Jetbrains Rider** (c:a 1.5gb)
    * `winget install jetbrains.rider`
* Installera **Visual Studio Build Tools 2026** (eller nyare) (c:a 200mb)
    * `winget install microsoft.visualstudio.buildtools`
* Starta Visual Studio Installer och lägg till komponenter:
    * **Windows 11 SDK** (c:a 2.6gb)
    * **.NET Framework SDK 4.8.1** (c:a 30mb)
    * **MSVC Build Tools for x86/x64 (Latest)** (c:a 2.1gb)

## Inställningar i Unreal

I Edit→Editor Preferences, gå till **Source Code** och ändra **Source Code Editor** till "Rider Uproject".
